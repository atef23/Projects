This program containts classes that hold information about a user's profile, as well as 
the user's status on the application. Users may create new profiles, login, and access information
about their status. Supports posting messages between users on walls, viewing messages, and
adding and viewing other users. Includes more robust features including checks for valid data
inputs and feedback. In addition, wall posts feature a love attribute which allows viewers to "love"
a post. There is a photo album feature in which users may store image source Strings for their account. 
The event and event calendar classes provide support for creating event calendars and storing events. 
ALl of the calendar support can be accessed through clicking the "Events" button. Friend adding is also now 
bi-directional.
